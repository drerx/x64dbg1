TraceIntoConditional[,ticnd]
Trace the program by stepping into, until specified condition is satisfied or maximun step count reached.

arguments
[arg1]: The condition of tracing. If it is evaluated to be a value other than 0, tracing finishes.

[arg2]: The maximum number of steps to trace. If it is not specified, the default value is 50000 steps.

result
This command does not set any result variables.
